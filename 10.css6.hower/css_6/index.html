Псевдоклассы это модификаторы селекторов

Используются для применения форматирования в зависимости от состояния

Параметры :hover начинают применяться к классу только при наведении мыши.

Модификатор :visited работает для уже посещенных ссылок

Порядок объявления :hover и :visited имеет значение

Псевдокласс :active работает в момент нажатия на объект

Псевдокласс :focus срабатывает при получении фокуса элементом (чаще полем ввода)

Псевдоклассы для работы со списком li, p: child

Несколько уточняющих псевдоклассов: block p:nth-child(3):hover
