position: relative позволяет сдвигать элемент
Дополнительные свойства: left, top, right, bottom
Значения можно указывать в процентах %
Значения могут быть отрицательные
z-index задает порядок наложения элементов, это число по умолчанию 0, со static не работает

.block__1: static
.block__2: relative(50px, 60px)
.block__3
